# Copilot Studio Deployment Guide

1. Create a new copilot in Copilot Studio.
2. Enable **Generative Answers** and select SharePoint as the knowledge source.
3. Map SharePoint fields to the KB schema (title, category, symptoms, resolution, keywords).
4. Import topic definitions from `copilot-config/topics/topic_definitions.json`.
5. Connect Power Automate flows for search and ticket creation.
